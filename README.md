# IndoScore Template

Template minimalis Expo + NativeWind + Tailwind CSS, JavaScript murni (no TypeScript).

## Stack
- Expo SDK 52 + Expo Router
- NativeWind v4 + Tailwind CSS v3
- JavaScript / JSX

## Cara Pakai
```bash
git clone https://github.com/galangMaulana01/indoreactscore.git
cd indoreactscore
npm install
npx expo start
```

Tekan `w` untuk web, `a` untuk Android.

## Build APK
```bash
eas build -p android --profile preview
```
